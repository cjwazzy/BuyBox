Please set your initial server buybox settings in config.yml.  Settings may be manipulated in game by using the built in commands.  See "/buybox help" for details.  CityName is for the name of the city/server that will be buying the items from players and will show up in info menus.  If your city/server has an iconomy account, purchases will also deduct from this account to pay players for their sales.
